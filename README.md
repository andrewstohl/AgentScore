# AgentScore

The Credit Rating Agency for the AI Agent Economy.

## About

AgentScore is a decentralized intelligence layer for agentic commerce, providing real-time trust scores for every service endpoint. Built as a Bittensor subnet, we combine predictive quality models with blind verification to create a robust trust scoring system for the emerging AI agent economy.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: Lucide React

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for static export and deploys automatically on Vercel via GitHub integration.
